Class: ContextMenu

.Client:UIElements > ContextMenu~ContextMenu()

Creates a contextmenu accessible by right clicking elements that have a defined one. The Context menu appears to the bottom left of the mouse position, unless the space is insufficient. Horizontally, it gets added to the bottom right of the mouse position, vertically, it gets placed flush with the bottom of the screen.
Contextmenus support clickable elements arranged in either a list or grid, unclickable elements include a title, a description and dividers. All elements can be placed anywhere.
